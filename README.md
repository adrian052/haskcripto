
# HaskCrypto

HaskCrypto is a cryptography library developed in Haskell. This library provides functions for encrypting and decrypting messages using various cryptographic algorithms, including the Caesar cipher, the VigenÃ¨re cipher, and more.

## Features

- Implementation of various cryptographic algorithms in Haskell.
- Easy to integrate into existing Haskell projects.
- Support for external contributions.

## Installation

To use HaskCrypto in your Haskell project, simply add `haskcrypto` as a dependency in your `.cabal` file and run `cabal install`.

```cabal
...
build-depends:       base >= 4.8 && < 5,
                     haskcrypto
...
```
